Exemple del Combinador de Descendents en CSS

Explicació del combinador de descendents:

El combinador de descendents ( ) s'utilitza quan en CSS escrivim dos selectors (com section i p) un al costat de l'altre amb un espai entre ells. Aquest combinador selecciona els elements que són descendents del primer element.

Relació entre section i p:

En aquest exemple, section p selecciona tots els elements <p> que es troben dins d'un element <section>. Això vol dir que només els paràgrafs que "viuen dins" d'un <section> tindran el fons blau; altres paràgrafs fora d'aquestes seccions no es veuran afectats per aquesta regla.

Exemple visual de la relació:

Aquest paràgraf és dins d'una secció.

Aquest paràgraf no és dins d'una secció.

Codi CSS:

section p { 
    background-color: blue;
}

Només el primer paràgraf tindrà el fons blau, perquè està dins del <section>. El segon paràgraf no serà afectat per l'estil, ja que està fora de la secció.

Així doncs, el combinador de descendents és útil per aplicar estils només als elements que es troben en una posició específica dins de la jerarquia de l'HTML.

Com utilitzar un atribut id en CSS

Quan tenim un element HTML amb un atribut id, com ara:

        <p id="remark">Aquest és un paràgraf especial</p>

Podem referir-nos a aquest identificador en CSS fent servir el símbol de la almoadilla (#) seguit de l'id. Això ens permet aplicar estils específics a aquest element concret. En aquest cas, l'estil per a l'element amb id="remark" es defineix com: 

#remark {
    color: darkgreen;
    font-weight: bold;
    background-color: lightyellow;
    padding: 5px;
    border: 1px solid green;
    border-radius: 3px;
}
